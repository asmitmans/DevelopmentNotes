# 03.1 — Proyecto Base (Spring Boot 4)

Este capítulo crea la base del proyecto
sobre la cual construiremos la seguridad JWT.

---

# Estado esperado al finalizar este capítulo

- Proyecto Spring Boot 4.0.2 generado.
- Dependencias básicas configuradas.
- Aplicación levanta correctamente.
- Endpoint de prueba accesible.
- Sin seguridad aún.

---

# Requisitos Previos

Ninguno.

Este es el punto de partida del módulo.

---

# 1. Generación del Proyecto

Usar Spring Initializr con:

- Project: Maven
- Language: Java
- Spring Boot: 4.0.2
- Packaging: Jar
- Java: 21 (o la versión definida en el proyecto)

Dependencias:

- Spring Web
- Spring Data JPA
- H2 Database
- Spring Security

---

# 2. Estructura Inicial del Proyecto

Al generar el proyecto, debe contener:

- Clase principal `SpringSecurityJWTDemoApplication`
- application.properties
- Dependencias en pom.xml

No agregamos aún:

- JwtService
- JwtFilter
- UserDetailsService personalizado

---

# 3. Verificar que la Aplicación Levanta

Ejecutar:

mvn spring-boot:run

La aplicación debe iniciar sin errores.

---

# 4. Endpoint de Prueba

Crear un controlador mínimo:

GET /api/ping

Debe devolver:

pong

Probar:

curl http://localhost:8080/api/ping

Resultado esperado:

pong

---

# Objetivo de este Capítulo

Tener una base funcional antes de introducir seguridad.

---
