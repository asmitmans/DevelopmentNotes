Ninguno.

---

# 1. Generación del Proyecto

Usar Spring Initializr con:

- Project: Maven
- Language: Java
- Spring Boot: 4.0.3
- Packaging: Jar
- Java: 21 (o la versión definida en el proyecto)

- Group: tu “dominio” invertido (identidad de organización).
    - Genérico: com.tuempresa / cl. tunombre / io.tuproyecto
    - Regla: estable, en minúsculas, sin espacios.
- Artifact: el nombre técnico del módulo (carpeta + nombre del .jar).
    - Genérico: (ejemplo) spring-security-jwt-demo o springsecurity-jwt-demo
    - Regla: corto, sin espacios, normalmente en kebab-case
- Name: nombre “humano” del proyecto (lo verás en el IDE).
    - Genérico: SpringSecurityJWTDemo
    - Regla: puede ser más bonito, no afecta tanto como artifact
- Description: texto libre (solo documentación/metadata).
    - Genérico: “Demo project for Spring Security using JWT”.
- Package name: paquete base donde quedará tu clase Application y desde donde
  Spring hace component scan.
    - Regla típica: Group + "." + Artifact (pero en formato de paquete)
    - Ejemplo: si Group=cl.agustin y Artifact=springsecurityjwtdemo →
      cl.agustin.springsecurityjwtdemo
- Packaging:
    - Jar (recomendado casi siempre): app Spring Boot ejecutable (java -jar).
    - War: solo si vas a desplegar en un servidor externo tipo Tomcat tradicional.

Ejemplo:
- Group: cl.tunombre
- Artifact: springsecurity-jwt-demo
- Name: SpringSecurityJWTDemo
- Package name: `cl.tunombre.springsecurity

Dependencias:

- Spring Web
- Spring Data JPA
- H2 Database
- Spring Security

---

# 2. Estructura Inicial del Proyecto

Al generar el proyecto, debe contener:

- Clase principal `SpringSecurityJWTDemoApplication`
- application.properties
- Dependencias en pom.xml

No agregamos aún:

- JwtService
- JwtFilter
- UserDetailsService personalizado

---

# 3. Verificar que la Aplicación Levanta

Ejecutar:

mvn spring-boot:run

La aplicación debe iniciar sin errores.

---

# 4. Endpoint de Prueba

Crear un controlador mínimo:

GET /api/ping

Debe devolver:

pong

Probar:

curl http://localhost:8080/api/ping

Resultado esperado:

pong
